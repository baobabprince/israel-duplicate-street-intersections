# מפגשי רחובות כפולים בישראל 🇮🇱

פרויקט לזיהוי וניתוח מפגשי רחובות כפולים בערים בישראל באמצעות נתוני OpenStreetMap.

## מה זה מפגש רחובות כפול?

כאשר שני רחובות נפגשים יותר מפעם אחת במרחק משמעותי (150+ מטר), זה יכול להעיד על תכנון עירוני מעניין, רחובות מקבילים, או מבנה רשת ייחודי.

## תכונות

- 🔍 **ניתוח אוטומטי** של 151 ערים בישראל
- 🗺️ **מפות אינטראקטיביות** עם OpenStreetMap
- 📊 **ייצוא לפורמטים מרובים** (HTML, CSV, JSON)
- 🎯 **סינון חכם** של שמות רחובות דומים וכפילויות
- 🌍 **מפה מאוחדת** של כל התוצאות בישראל

## תוצאות

נמצאו **684 מפגשי רחובות כפולים** ב-**63 ערים** בישראל!

### ערים מובילות:
1. **ירושלים** - 91 מפגשים (מרחק מקסימלי: 2,046 מטר)
2. **באר שבע** - 59 מפגשים
3. **חיפה** - 38 מפגשים
4. **אשקלון** - 33 מפגשים
5. **ראשון לציון** - 31 מפגשים

## שימוש

### ניתוח עיר בודדת:
```bash
python find_duplicate_intersections.py "ירושלים"
```

### עם סינון מרחק:
```bash
python find_duplicate_intersections.py "תל אביב-יפו" 200 1000
```

### ניתוח כל הערים:
```bash
python run_all_cities.py
```

### יצירת קובץ מאוחד:
```bash
python create_unified_results.py
```

## אלגוריתם

1. **הורדת נתונים** מ-OpenStreetMap Overpass API
2. **זיהוי צמתים** - מציאת כל הנקודות שבהן רחובות נפגשים
3. **חיפוש כפילויות** - זיהוי זוגות רחובות שנפגשים יותר מפעם אחת
4. **סינון חכם**:
   - הסרת שמות דומים (למשל "גולדה מאיר" ו"שדרות גולדה מאיר")
   - שמירת מפגש אחד לכל זוג רחובות (המרוחק ביותר)
5. **ייצוא** למפות אינטראקטיביות וקבצי נתונים
